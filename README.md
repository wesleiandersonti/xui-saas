# XUI-SAAS

Plataforma SaaS para automação e gestão de operações baseadas em XUI-One.

## Objetivos
- Conectar em instâncias XUI-One via MariaDB (sem instalar XUI para o cliente)
- Automação de clientes, testes, renovações e bloqueios
- Notificações WhatsApp/Telegram
- Integrações n8n por cliente
- VOD com TMDB e marketing (jogos do dia)
- Multi-revendedor + comissionamento
- Backups automáticos

## Infra
- Docker / Docker Compose
- Cloudflare Tunnel (exposição segura)
